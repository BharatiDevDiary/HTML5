## 1. Basic Table Structure

- The `<table>` tag is used to create tables in HTML.
- A table is made up of rows (`<tr>`) and columns (`<td>`).

```html
<table border="1">
  <tr>
    <td>Row 1, Column 1</td>
    <td>Row 1, Column 2</td>
  </tr>
  <tr>
    <td>Row 2, Column 1</td>
    <td>Row 2, Column 2</td>
  </tr>
</table>
```
